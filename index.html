Särmä

Särmä simplifies Finnish military conscription preparation by tracking deadlines, tasks, and key milestones. Features include countdown timers, Cooper test tracking, packing lists, and budget management. Built with SwiftUI as a native iOS app with offline support. Achieved 7.6% App Store conversion rate from organic search with zero paid acquisition.

Yo-StudySuite

Yo-StudySuite is a electron based desktop study app, it is made for students studying /preparing for their matriculation exams. It includes to-do list, notes, cross subject ai help, flashcard integration and analytics. It minimizes the need for context switching.

Essaykit LaTeX package

Essaykit is latex package made for people startinr out with writing essays and academics papers with LaTeX. Version 2.0 was released in early 2026.The purpose of the package is to provide easy to read / write macros with more describtive syntax for common essay components such as figures and styled boxes.
